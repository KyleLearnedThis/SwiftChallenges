# SwiftChallenges

A personal Swift practice repository covering data structures, algorithms, and coding challenges from platforms like LeetCode, HackerRank, CodeSignal, and Meta. Each solution has a corresponding test file under `SwiftChallengesTests/`.

---

## Data Structures (`DS/`)

Implementations of core data structures with generics.

| Area | Implementations |
|------|----------------|
| **Cache** | LRU Cache, LFU Cache |
| **Dictionary** | Ordered Map |
| **Graph** | Vertex, Edge, Graph (BFS, DFS), Dijkstra's shortest path |
| **List** | Linked List, Node |
| **Numbers** | Sieve of Eratosthenes |
| **Queue** | Queue, Stack, Priority Queue (min-heap), Maximum Frequency Stack |
| **Sort** | Bubble Sort, Heap Sort, Merge Sort, Quick Sort |
| **Tree** | Binary Search Tree, Trie |

---

## Lab (`Lab/`)

Practice problems organized by category and source platform.

### Classics
Anagram, Palindrome, FizzBuzz, TwoSum, Valid Parentheses, Longest Common Prefix, Stock Trading, Combination Sum, Top K Elements, and more.

### Collections / Matrices
MineSweeper, Battleship, Box Blur, Spiral Traversal, Number of Islands, Reshape Matrix, Unique Paths, Diagonal Sum Difference, and more.

### Numbers / Arrays
Two Sum, Median of Two Sorted Arrays, Daily Temperatures, Array Permutations, Binary Search, Avoid Obstacles, Deposit Profit, and more.

### Strings / Text
Decode Message, Roman Numerals, Valid Variable Name, Reverse Parentheses, IP Checker, Goal Parser, Keyboard Row, and more.

### Trees (BST)
Inversion, Path Sum, Lowest Common Ancestor, Max Height, Balance BST, Symmetric Check, Merge Two BSTs, and more.

### Linked Lists
Detect Cycle, List Reverse, Sum of Two Lists.

### Dynamic Programming
Fibonacci, Climb Stairs.

### Graph Problems
Destination City, Airport Route.

### Bit Manipulation
Number to Binary, Single Number.

### HackerRank
Beautiful Days, Count Valleys, Divisible Sum Pairs, Migratory Birds, Sock Match, Utopian Tree, and more.

### Meta
Rotary Lock, Rotational Cipher, BST Average Level, Encrypted Words, Partition Equal Sum, Revenue Milestones, Queue Removals, and more.

### Swift Exploration
Closures, Combine framework, networking (URLSession), OOP patterns (protocols, generics), collection transforms, regex, and type conversions.
